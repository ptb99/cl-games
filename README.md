# cl-games
Various Common-Lisp games and graphics based on SDL


### ball-demo.lisp

The classic bouncing-ball demo to test that graphics display properly


### show-image.lisp

Basic test to load a .BMP file and then exit


### flickr-images.lisp

Test Flickr retrieval code, defaulting to keyword search


### pic-puzzle.lisp

CL version of my Python puzzle code.  Retrieves a Flickr image and asks you
to rearrange the tiles to form the proper image.


### gp-images.lisp

Experiment with Genetic Algorithms by replicating the original Karl Sims
image generation,  Highlight the best images and then evolve them to
populate the rest and (hopefully) improve the quality.
